# ML Trading Backtester

Профессиональная система для бэктестинга и машинного обучения на финансовых рынках.

## 🚀 Возможности

- **Загрузка данных** из MetaTrader 5 с кэшированием
- **70+ технических индикаторов** (трендовые, осцилляторы, объемные, волатильность, паттерны)
- **Автоматический отбор признаков** (удаление коллинеарных, отбор по важности)
- **Множество моделей ML**: Random Forest, XGBoost, LightGBM, LSTM, GRU, Transformer
- **Ансамбли и гибридные модели** (CNN-LSTM, Attention, Stacking)
- **Профессиональный бэктестинг** с комиссиями, спредами, риск-менеджментом
- **30+ метрик** производительности и риска (Sharpe, Sortino, VaR, CVaR и др.)
- **Визуализация** всех результатов
- **HTML отчеты** с графиками и таблицами
- **Telegram уведомления** о сделках и результатах
- **Реестр моделей** для сравнения и выбора лучших
